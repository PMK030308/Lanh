🎵 NHẠC NỀN (nhiều bài)

Web phát nhiều bài theo danh sách. Các file nhạc nằm ngay trong thư mục
"public" này. Hiện đang có:

    aingaoianh.mp3     -> "Ai Ngại Ngùng Anh"
    Mattroicuaem.mp3   -> "Mặt Trời Của Em"
    1thanh2.mp3        -> "1 Thành 2"
    votay.mp3          -> "Vỗ Tay"

- Bài đầu TỰ PHÁT khi mở trang (nếu trình duyệt chặn, nhạc bật khi
  chạm/bấm lần đầu). Hết bài sẽ tự sang bài kế, hết danh sách thì quay lại đầu.
- Nút tròn ♫ góc phải trên: phát/tạm dừng. Nút nhỏ bên cạnh: mở danh sách
  để chọn bài + nút chuyển bài trước/sau.
- Trang "Playlist" (trang 8) cũng bấm chọn bài được, bài đang phát sẽ sáng lên.

THÊM / ĐỔI BÀI:
Mở file  src/data.js , sửa danh sách TRACKS (thêm dòng { file, title }).
Nhớ chép file .mp3 tương ứng vào thư mục public/ này.
